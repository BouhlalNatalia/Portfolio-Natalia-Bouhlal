Проекты

My Pet Shop

GitHub Live Demo

Разработала и подготовила к релизу полноценное E-commerce решение для зоомагазина. В ходе работы спроектировала компонентную архитектуру на React, обеспечила бесшовную навигацию и реализовала сложную логику управления товарными запасами в корзине. Особое внимание уделила чистоте кода и соответствию современным стандартам веб-разработки.

React Redux Toolkit Axios React Router Dom Formik / Yup REST API
State Management: Внедрена технология Redux Toolkit для управления глобальным состоянием, что позволило синхронизировать корзину, избранное и фильтры между всеми страницами приложения.
API Integration: Оптимизирован процесс получения данных с бэкенда через Axios, настроила систему перехватчиков (interceptors) и автоматизировала обработку ошибок и состояний загрузки.
Routing & Navigation: Реализована многоуровневая система маршрутизации с помощью React Router Dom, включая динамические сегменты для карточек товаров и защищенные переходы.
Form Automation: Автоматизирован процесс сбора и проверки данных покупателя, интегрировав Formik и Yup, что свело к минимуму риск ошибок при оформлении заказа.
Performance & UX: Проведен рефакторинг компонентов для предотвращения лишних рендеров и адаптирован интерфейс под все типы устройств, обеспечив высокую скорость работы приложения.

Events Platform

GitHub Live Demo

Многостраничная платформа для поиска и бронирования мероприятий. Реализована сложная система динамической фильтрации событий по категориям и датам, а также спроектирован интуитивно понятный поьзовательский путь от главной страницы до выбора конкретного ивента.

HTML5SASS/SCSSJavaScriptCSS3GulpLeaflet API
Динамическая фильтрация: Реализована логика отбора мероприятий по 4 критериям: типу, формату (дистанция), категории и дате.
Архитектура БЭМ: Использована строгая методология именования для создания независимых и переиспользуемых UI-компонентов.
Workflow & Сборка: Автоматизация процессов с помощью Gulp и использование препроцессора SASS для эффективного управления стилями.
Интерактивная карта: Интеграция Leaflet API для визуализации локаций мероприятий и улучшения UX-взаимодействия.
SEO & Доступность: Оптимизированная семантическая верстка (HTML5) для корректной индексации и высокой доступности интерфейса.
Responsive Design: Глубокая проработка адаптивности, обеспечивающая стабильную работу фильтров и сеток на смартфонах и десктопах.

HOTSAUCE Conference

GitHub Live Demo

Профессиональный адаптивный лендинг для технологической конференции HOTSAUCE. Проект реализован как полноценный кейс с акцентом на чистую архитектуру кода, сложную геометрическую верстку и интерактивные пользовательские интерфейсы. Особое внимание уделено UX-деталям: плавной навигации, информативным секциям цен и удобному мобильному представлению.

HTML5CSS3JavaScriptSASS (SCSS)Gulp
Архитектура БЭМ: Реализована строгая модульная структура кода, обеспечивающая полную независимость компонентов и чистоту стилей.
Layout (Grid & Flex): Спроектирована сложная адаптивная сетка для секций спикеров и расписания, корректно работающая на всех типах устройств.
Креативная стилизация: Применены техники работы с псевдоэлементами (::before/::after) и SVG для создания уникального эффекта «перфорированного билета».
Интерактивный UI: Интегрированы FAQ-аккордеоны и система уведомлений на Vanilla JavaScript с использованием плавных CSS-анимаций.
Полный адаптивный цикл: Произведена оптимизация интерфейса под 5 ключевых разрешений (от 390px до 1920px).
Workflow & SEO: Использованы SASS и Gulp для автоматизации сборки, а также семантическая разметка HTML5 для SEO-оптимизации.
